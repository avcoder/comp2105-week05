CSS animation

  • 1. Define the animation using keyframes
  • 2. Assign it to a specific element (or elements)

1. Define the animation using @keyframes

Keyframes are a list describing what should happen over the course of the animation


            @keyframes name-this {

            }

It's important to name your keyframe animations, otherwise you can never use it.

See the Pen animation - from to by A.V. ( @avcoder) on CodePen.

1a) Define @keyframes structure with name 

                @keyframes insert-name {

                }

See the Pen animation - from to by A.V. ( @avcoder) on CodePen.

1b) Declare visual changes via
from to OR 0% 100% 

                  @keyframes bobble {
                    from { transform: translateY(0); }
                    to { transform: translateY(100px); }
                  }

2. Assign animation name to a specific element(s)


          #elementID {
               animation: bobble 1s;
          }

Also include animation-duration

See the Pen animation - from to by A.V. ( @avcoder) on CodePen.

2. Assign it to a specific element using keyword animation 

                    @keyframes bobble {
                      from { transform: translateY(0); }
                      to { transform: translateY(100px); }
                    }

                    #hexagon {
                      animation: bobble 1s;
                    }
